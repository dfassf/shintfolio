백엔드 개발자

초기 설계부터 운영까지 경험한
4년차 Node.js 백엔드 개발자

4년차 백엔드 중심의 풀스택 개발자로, NestJS 기반의 서버 아키텍처 설계와 도메인 모델링에 강점을 가지고 있습니다. 인증서 발행량 30만건, 카카오톡 채널 친구 1만 5천명에 달하는 서비스의 초기 개발부터 운영, 리팩토링, 마이그레이션까지 전 주기 경험을 보유하고 있으며, 기술 부채 해소와 구조적 개선을 통해 시스템 안정성과 비용 효율을 실질적으로 개선한 경험이 있습니다. 도메인 요구사항을 빠르게 파악하고, 기획자 및 외부 서비스 담당자와의 긴밀한 협업을 통해 문제를 구체화하고 해결해나가는 데 익숙합니다. 유지보수성과 생산성을 동시에 고려한 설계와 구현, 그리고 사용자 경험까지 아우르는 사고로 팀과 서비스에 기여해왔습니다.

연락하기

소개

백엔드 개발에 풀스택 경험을 활용해요.

프론트엔드 중심 풀스택 개발자로 커리어를 시작했지만 서티라이프에서 백엔드 설계 및 개발을 담당하게 되며 백엔드에 더 큰 전문성을 가지게 되었습니다. 약 2년간 의료 인증서 발행 서비스의 초기 개발부터 참여하여 30만 건 이상의 인증서 발행과 1만 5천 명 이상의 카카오 로그인 회원을 보유한 안정적인 서비스가 될 때까지 운영에 참여하였습니다.

NestJS, MongoDB, TypeScript를 주력으로 사용하며, 클린 코드와 유지보수성 높은 아키텍처 설계에 중점을 둡니다. 프론트엔드 개발 경험을 통해 API 설계 시 화면단에서의 데이터 사용성과 효율성을 고려할 수 있는 것이 저의 강점입니다.

기획자와 소통하며 기능의 사용성과 쓰임새에 대해 늘 고민하는 것이 습관화 되어있으며, 팀 내에서 기술적 의사결정과 문제 해결에 적극적으로 참여하는 개발자가 되고자 합니다.

백엔드 중심 풀스택 개발자

30만 건 이상의 인증서를 발행한 의료 인증서 플랫폼으로, 병원과 환자 간 디지털 의료 인증서를 제공하며 제조사와 환자의 연결고리를 만들어 주는 서비스입니다. 초기 프론트엔드, 백엔드 개발부터 안정적인 운영까지 핵심 개발자로 참여했습니다.

역할 및 기여

  • 병원 관리자, 제조사 관리자, 인증서 수신화면, 어드민 페이지 개발 및 유지보수
  • 커스텀 인증서 발행/템플릿 관리, 이벤트 및 전자계약 등 주요 도메인 풀스택 개발 및 고도화
  • MongoDB 기반의 도메인 모델링, 인증 및 알림 관련 API 개발 및 자동화 작업 수행
  • 알림톡 관련 기능 유지보수 및 리팩토링
  • 카카오클라우드 → Azure 이전 작업 지원

성과

  • 자체 전자계약 기능 구현으로 연간 약 400만 원 외부 서비스 비용 절감
  • 인증서 발행량 약 30만 건 달성, 카카오 채널 유입자 수 15,000명 확보
  • Azure Static Web App 및 Vite 도입을 통해 연간 약 60만 원 인프라 비용 절감
  • 무한 계층 카테고리 구조 도입으로 인증서 관리 유연성 개선
  • 지속적인 리팩토링을 통해 유지보수성 향상 및 오류 발생률 감소

주요 경험 및 문제 해결 사례

  • 서비스 초기 개발 및 시스템 안정화
    • 기존 외주팀이 구현한 Express 기반 프로젝트를 NestJS로 마이그레이션하여 가독성과 확장성 향상, 타 개발자 협업 환경 개선
    • Bootstrap 기반 스타일 충돌 문제를 직접 구현 방식으로 대체하여 운영 환경 안정성 확보
    • 웹팩 기반 레거시 관리자 프로젝트를 Vite로 전환, 빌드 시간 약 40% 단축
  • 핵심 기능 구조화 및 운영 효율 개선
    • 고정된 인증서 구조를 무한 카테고리 기반 설계로 변경, 다양한 케이스에 유연하게 대응
    • 외부 API 장애 영향을 줄이기 위해 내부 DB 우선 조회 및 서비스 제공자 분기 처리, 연간 약 400만원 비용 절감
    • 카카오 개인정보 수집 항목 재정비 및 심사 통과, 신규 로그인 기능 도입으로 플러스 친구 약 1.5만 명 유입
  • 레거시 개선 및 리팩토링
    • 단일 책임 원칙 기반으로 인증서 발행 로직 기능 분리, 사이드 이펙트 제거 및 QA 작업 간소화
